◆ push_ml_arena()

ML_Status push_ml_arena ( void ** ptr,
ml_arena * arena,
u64 size )

Allocate size bytes from the arena (bump allocation).

Allocations are aligned to ARENA_ALIGN.

Behavior:

  • On success, *ptr receives the allocated pointer and the arena position advances.
  • If size == 0, *ptr is set to NULL and the function returns ML_OK.
  • If there is not enough space, returns ML_OUT_OF_MEMORY and does not modify the arena.
Parameters
ptrOutput pointer to receive the allocated address.
arenaArena to allocate from.
sizeNumber of bytes to allocate.
Returns
ML_OK on success.
ML_INVALID_ARGUMENT if ptr is NULL, or arena is NULL, or arena->base is NULL.
ML_OUT_OF_MEMORY if the arena does not have sufficient remaining capacity.
